# VPP_Latency_Measurements
Data from the mobile power supply unit (PSU) VPP fast frequency reserve activation latency measurement data

Files are in MS Excel.xlsx format

They contain the delay measurement data for the actual measurements for the VPP delay article. The files are
- 'PSU_Latency_ Vertiv_CLEAN.xlsx' contains Vertiv PSU delay maesurements and the histogram plot from the delay
- 'Acuvim_Latency_Measurement_CLEAN' contains Acuvim frequency measurement delay maesurements and the histogram plot from the delay
- 'Latency_ Measurement_CLEAN' contains one instance of the activation delay with respect to time in x-axis and the plot from the delay

A public DOI.org for this repository is https://doi.org/10.5281/zenodo.19781922
